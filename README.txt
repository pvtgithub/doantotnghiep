-----Hướng dẫn cài đặt chương trình website-----
Các phiên bản sử dụng trong chương trình:
1. Xampp Control Panel (version: 3.3.0)
2. Laravel Framework (version: 8.83.23)
3. PHP (version: 7.4.27)
4. Composer (version: 2.2.7)
Hướng dẫn cài đặt:
Bước 1: Download code theo đường link https://drive.google.com/file/d/1-OiEgTWzCL95IWel_5k6WwBixtltsLMW/view?usp=share_link và giải nén.
Bước 2: Tại đường dẫn thư mục code mở cmd và chạy dòng lệnh php artisan serve.
